Format Sheet yang Diharapkan

Sistem otomatis mendeteksi kolom dari nama sheet. Berikut format kolom yang dikenali dari file report-*.xlsx:

Sheet: Chat / Chats

Data Percakapan

Berisi data sesi chat per agen per hari.

  • Date — Tanggal (YYYY-MM-DD)
  • Agent / Assignee — Email agen
  • New Conversations — Chat baru
  • Open Conversations — Chat aktif
  • Avg First Response Time — Waktu respon (menit)
  • Avg Duration — Rata-rata durasi chat (detik)
  • Messages Sent — Jumlah pesan dikirim
Sheet: Invoice / Invoices

Data Invoice

Berisi data tagihan yang dibuat per agen per hari.

  • Date — Tanggal invoice
  • Agent / Assignee — Email agen
  • Count / Total — Jumlah invoice
  • Nominal / Amount — Nilai (Rupiah)
  • Status — paid / unpaid / expired
Sheet: Agent / Agents / Online

Data Waktu Online

Berisi akumulasi jam kerja online per agen.

  • Agent / Email — Email agen
  • Online Hours / Jam Online — Total jam
  • Assigned — Jumlah pelanggan dialokasikan

💡 Tip: Sistem bekerja fleksibel — nama kolom tidak harus persis sama, cukup mengandung kata kunci yang relevan (case-insensitive). Bila sheet tidak ditemukan, data dari kolom Excel yang ada tetap akan diekstrak secara otomatis.
